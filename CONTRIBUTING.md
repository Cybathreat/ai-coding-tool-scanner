# Contributing to AI Coding Tool Scanner

Thank you for contributing to AI Coding Tool Scanner.

## Development Setup

```bash
git clone https://github.com/cybathreat/ai-coding-tool-scanner.git
cd ai-coding-tool-scanner
pip install -e ".[dev]"
```

## Running Tests

```bash
pytest
```

## Pull Request Guidelines

1. All PRs must include tests
2. Keep test coverage >= 75%
3. Follow PEP 8 style
4. One feature per PR

## Reporting Issues

Include:
- Tool version
- Python version
- Steps to reproduce
- Expected vs actual behavior
